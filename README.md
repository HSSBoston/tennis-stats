# Event-Driven Gain in Expectancy (EDGE): A Probablistic Metric for Comprehensive Player Performance in Tennis

Traditional tennis statistics, such as aces, double faults, winners, and unforced errors, provide intuitive box-score summaries of player performance. However, these primitive metrics evaluate isolated aspects of performance rather than summarizing a player's overall performance. As a result, comparing individual players often requires tables or other visualizations that combine multiple separate metrics. This makes interpretation less direct.
